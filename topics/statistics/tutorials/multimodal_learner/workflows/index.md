# Workflows

This directory contains Galaxy workflows for the Multimodal Learner tutorial.

## Main Workflow

The main workflow (`main_workflow.ga`) includes:
1. Data upload and validation
2. Multimodal Learner model training
3. Model evaluation and visualization

## Using the Workflow

1. Import the workflow into your Galaxy account
2. Upload your data (CSV and ZIP files)
3. Run the workflow with default parameters or customize as needed
4. Review the output HTML report and model file

## Workflow Tests

The `main_workflow-tests.yml` file contains automated tests to ensure the workflow runs correctly.
